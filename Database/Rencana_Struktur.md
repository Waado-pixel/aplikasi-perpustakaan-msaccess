# Rencana Struktur Database

Database ini akan menggunakan Microsoft Access (.accdb) dengan tabel sebagai berikut:

### 1. Modul Retail
- **Tabel_Barang**: ID, Nama_Barang, Harga, Stok.
- **Tabel_Penjualan**: ID_Transaksi, Tanggal, Total_Harga.

### 2. Modul Perpustakaan
- **Tabel_Buku**: ID_Buku, Judul, Pengarang, Status.
- **Tabel_Anggota**: ID_Anggota, Nama, Alamat.
