# Aplikasi Perpustakaan Desktop

Aplikasi perpustakaan berbasis desktop menggunakan Microsoft Access.

## Fitur
- CRUD Data Buku
- CRUD Data Anggota
- Menu Utama

## Teknologi
- Microsoft Access
- VBA

## Cara Menjalankan
1. Download file `perpustakaan.accdb`
2. Buka menggunakan Microsoft Access
3. Jalankan form `frm_menu`
