## Cara Menjalankan Aplikasi

Ikuti langkah-langkah berikut untuk menjalankan aplikasi di komputer Anda:

1. **Download Project:**
   - Klik tombol hijau **"Code"** di bagian atas, lalu pilih **"Download ZIP"**.
   - Ekstrak file ZIP tersebut di folder komputer Anda.

2. **Buka File Database:**
   - Pastikan Anda sudah menginstall **Microsoft Access** (Minimal versi 2013 atau yang lebih baru).
   - Klik dua kali pada file `perpustakaan.accdb`.

3. **Aktifkan Konten (Penting):**
   - Jika muncul peringatan *"Security Warning: Macros have been disabled"*, klik tombol **"Enable Content"**. Ini diperlukan agar fitur VBA (coding) di dalam aplikasi bisa berjalan.

4. **Login/Masuk:**
   - Gunakan username dan password yang sudah ditentukan (Jika ada).
